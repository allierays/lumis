# YouTube Description

## Instructions

Write a YouTube video description that serves two audiences: the algorithm and the human viewer. The description must feel like it was written by the creator, not assembled by a tool.

### Step 0: Gather Context

Ask the user for:
1. **What is the video about?** (topic, angle, key takeaway)
2. **Who is it for?** (audience)
3. **What is the one thing you want viewers to do after watching?** (primary CTA: visit a site, subscribe, download something, watch another video)

If the user provides a script, voiceover text, or video file, use that as the source material instead of asking.

Read Voice.md from the vault if it exists. Match the creator's tone.

### Step 1: Write the Hook (First 150 Characters)

This is the most important part. These characters appear in YouTube search results, Google search results, and above the "Show more" fold.

Rules:
- Include the primary keyword in the first 25 words
- Answer: "What is this video about?" and "Why should I watch?"
- Write like a hook, not a summary
- Never start with "Hey everyone", "Welcome back", "In this video I..."
- Never start with the channel name
- Be specific about the outcome or value

Present the hook to the user for approval before continuing.

### Step 2: Write the Full Description

Structure (in this order):

```
[HOOK: 2-3 lines] — above the fold
  Primary keyword in first 25 words
  Specific outcome or value promise

[VALUE: 3-5 lines] — what the viewer gets
  Key points covered in the video
  Natural keyword variations (not stuffing)

[PRIMARY CTA + LINK] — one main action
  One line explaining why to click
  The link

[WATCH NEXT] — internal link
  Link to a related video on the channel

[TIMESTAMPS] — if video is over 5 minutes
  0:00 Introduction
  X:XX Keyword-rich section labels

[RESOURCES] — tools, links, references mentioned
  Each link gets a one-line reason to click
  No bare URLs without context

[HASHTAGS] — 3-5 at the end
  1 broad, 2 mid-tier, 1-2 niche
```

Rules:
- Total length: 150-300 words (800-1,500 characters)
- Use line breaks between every section
- Format for mobile first
- Include the primary keyword 3-4 times naturally, never more
- Every link needs a one-line reason to click
- One primary CTA per description, placed after the value section
- Match the description to actual video content. If the video doesn't cover it, don't mention it

### Step 3: Apply Humanizer Rules

Before presenting the final description, check for and remove:
- AI vocabulary (delve, landscape, crucial, leverage, robust, innovative, tapestry, multifaceted, holistic, comprehensive, pivotal, groundbreaking, paradigm, foster, encompasses)
- Significance inflation ("stands as a testament to", "reflects broader themes of")
- Filler phrases ("It's worth noting that", "Interestingly", "In today's X")
- Em dash overuse. Use commas, colons, or periods
- Sycophantic language
- Promotional/brochure tone
- Generic positive conclusions

The description should sound like the creator talking, not a marketing department.

### Step 4: Present and Revise

Show the full description to the user formatted exactly as it would appear on YouTube. Ask if anything needs adjusting.

If the user provides a video URL, offer to write timestamps based on the content.

### What NOT to Do

- Never keyword stuff (repeating the same phrase 5+ times)
- Never use more than 5 hashtags (YouTube ignores ALL hashtags if you exceed 15, and more than 5 looks spammy)
- Never write a wall of text with no line breaks
- Never copy-paste identical descriptions across videos
- Never include irrelevant hashtags for reach
- Never bury the primary link at the bottom
- Never use clickbait that doesn't match the video content
- Never write "Like and subscribe!" as filler
- Never use the rule of three ("informative, engaging, and transformative")
- Never use negative parallelisms ("It's not just X; it's Y")
